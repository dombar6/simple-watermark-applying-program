# Simple watermark applying program
A simple program that applies a watermark text on an image.
# Dependencies
For the program to work the Pillow library is need, you can see instalation steps here:  
[https://pillow.readthedocs.io/en/latest/installation/basic-installation.html](https://pillow.readthedocs.io/en/latest/installation/basic-installation.html)  
# Starting the program
To start the program have the Pillow library installed and run the main.py script in VS Code or pyCharm.
